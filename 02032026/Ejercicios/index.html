🔢 Bloque 1: Ejercicios Básicos

💰 Calculadora de Propinas

Cuenta: $150 | Propina: 10%

📝 Ver código 

// Código del ejercicio
let cuenta = 150;
let propina = cuenta * 0.10;
let total = cuenta + propina;
console.log("Total a pagar: $" + total);

✏️ Concatenación de Nombres

Nombre: EjemploNataly | Apellido: EjemploReyes

📝 Ver código 

let nombre = "EjemploNataly";
let apellido = "EjemploReyes";
let nombreCompleto = nombre + " " + apellido;
console.log(nombreCompleto);
console.log("Tipo de dato: " + typeof nombreCompleto);

🔢 El Contador

puntos = 10 → +5 → -2

📝 Ver código 

let puntos = 10;
puntos = puntos + 5;
puntos = puntos - 2;
console.log("Puntos finales: " + puntos);

📅 Conversor Años → Días

📝 Ver código 

let edad = 25; // o la edad ingresada
let totalDias = edad * 365;
console.log("Has vivido aproximadamente " + totalDias + " días");

🍪 Resto de División

17 galletas entre 3 amigos

📝 Ver código 

let galletas = 17;
let amigos = 3;
let sobrantes = galletas % amigos;
console.log("Sobran " + sobrantes + " galletas");
